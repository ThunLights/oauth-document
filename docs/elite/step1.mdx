# サイトにOauthURLを配置

Oauthを扱うためにサイトにログイン用リンクを貼りつけていきます。

以下のようなaタグを入れることによってThunLights Oauthを扱う事が出来るようになります。
```html
<a href="https://thunlights.com/oauth?site={DOMAIN}">LOGIN</a>
```

**※\{DOMAIN\}の部分は、使う予定のドメインを入れてください (例: example.com)**

## クリックするとどうなるの?

クリックをするとthunlightsのoauth用のページに飛びます。

そのページではあなたのドメインと連携するかどうかの旨を問われ、
連携するボタンを押した場合あなたが事前に指定したリダイレクト先にcodeクエリを引き下げた状態でリダイレクトされます。

Step2ではそのコードの扱いについて解説します。
