# Add Tags to HTML

By including an anchor tag like the one below, you can handle ThunLights Oauth.

```html
<a href="https://thunlights.com/oauth/{applicationId}">LOGIN</a>
```

The code required in exchange for the access token will be passed as a code query to the pre-specified redirect URL.
