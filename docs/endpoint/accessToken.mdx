# /check/accessToken/$accessToken

This API can be used to convert access tokens into user information.

Example

```ts
const accessToken = "";

async function main() {
	const response = await fetch(`https://oauth.thunlights.com/check/accessToken/${accessToken}`, {
		method: "POST"
	});

	if (response.ok) {
		console.log("-------DATA-------");
		console.log(await response.json());
	}
}

main();
```

Response

```json
{
	"id": "ID",
	"txtId": "TXTID",
	"email": "EMAIL",
	"displayName": "DISPLAYNAME"
}
```
