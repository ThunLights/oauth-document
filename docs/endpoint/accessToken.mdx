# /check/accessToken/$accessToken

このAPIを使う事によってアクセストークンをユーザー情報に変換できます。

Example
```ts
const accessToken = "";

async function main() {
    const response = await fetch(`https://oauth.thunlights.com/check/accessToken/${accessToken}`, {
        method: "POST",
    });

    if (response.ok) {
        console.log("-------DATA-------");
        console.log(await response.json());
    }
}

main();
```

Response
```json
{
    "id": "ID",
    "txtId": "TXTID",
    "email": "EMAIL",
    "displayName": "DISPLAYNAME"
}
```
