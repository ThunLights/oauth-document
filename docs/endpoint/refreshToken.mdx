# /check/refresh/$refreshToken

This API can be used to reissue access tokens.

Example
```ts
const id = 0;
const refreshToken = "";
const secret = "";
const domain = encodeURIComponent("");
const url = `https://oauth.thunlights.com/check/refreshToken/${refreshToken}`;

async function main() {
    const response = await fetch(`${url}?secret=${secret}&domain=${domain}&id=${id}`, {
        method: "POST",
    });

    if (response.ok) {
        console.log("-------DATA-------");
        console.log(await response.json());
    }
}

main();
```

Response
```json
{
    "accessToken": "ACCESS_TOKEN"
}
```
